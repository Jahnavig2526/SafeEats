
 🔹 1. Project Title + One-Line Description
SafeEats – AI-powered app to detect food allergens and recommend safe dishes
 🔹 2. Problem
People with food allergies can’t easily understand restaurant menus and risk unsafe food.
🔹 3. Solution
App that:
* stores user allergies
* uses AI to detect ingredients
* shows **safe / unsafe dishes*
🔹 4. Features
* Allergy profile
* AI ingredient + allergen detection
* Menu scanner (image/text)
* Safe food recommendations
🔹 5. Tech Stack
* Frontend: React
* Backend: Node.js / FastAPI
* DB: PostgreSQL
* AI: LLM / NLP
🔹 6. How it Works
Menu → AI → Ingredients → Allergens → Compare with user → Resul
